## 프로젝트 개요

옷을 중점으로 판매하는 쇼핑몰 사이트를 Spring Framework를 이용하여 구현한다. 회원가입/로그인/로그아웃 등 기본적인 회원관리 기능과 옷의 종류 (상의,하의 등)을 카테고리로 구별하여 관리한다. 쇼핑몰 회원은 올라간 상품에 대한 댓글을 작성하거나 본인 카트에 담는 기능 등을 구현 한다.
